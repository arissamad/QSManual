Configuring the Homework Module

Configuring the Homework Module is necessary before homework can be assigned. This process must be completed by an administrator. The first steps to activate the Homework Module:


  1. Click on Features Option at top of page.
  2. Check box next to Homework option.
  3. Click on Configure next to check box.

This pops up a window which allows administrators to list personnel who are authorized to manage Homework. In addition to teachers, other school personnel may be authorized to enter homework details on behalf of teachers. Administrators can also authorize select users to run reports.




To complete configuration of Homework Module:


  1. Check box if you want teachers to manage their homework assignments including attachments.
  2. List personnel who can globally enter homework details on behalf of teachers in box provided.
  3. List personnel who can run reports.
  4. Click the save button.

Homework reports are accessible in Reports Module.



Assign Homework

Access the Homework Module by choosing "Homework" link at top of page. The Homework page shows an overview of homework that has been assigned and are listed in numerical order by grade then alphabetical by subject.



Add New Homework Assignment

To add homework for a subject click the Add/Edit Notes link next to subject name. The grade/subject expands displaying an area to enter new homework assignment that looks like this:



Fill in Description, Date Assigned, and Due Date. Click green Add Note button to save new homework assignment. NOTE: Attachments are uploaded after new homework assignment is added.


Adding an attachment:

Once a new homework assignment is added an attachment maybe uploaded. Find newly added assignment and chooose Attach File link, as shown below. Pick file to attach and click the Choose button. The file may take a few moment to upload depending on it's size. Students and parents are able to download the attachment at their respective portals.

Caution: Attachments cannot be wholly deleted, only replaced with another attachment.




Edit/Delete Homework Assignments

Homework assignments can be edited or deleted.

Edit Homework by clicking on Add/Edit link next to homework assignment. Click on the red Edit button in the assignment to be edited. This opens an Edit Note box as pictured below. Changes may be made to Description, Assigned Date, and Due Date. Remember to click red Save button to keep any changes made.




Attachments may be replaced but never deleted. Replace an attachment by clicking the Attach Different File link, as highlighted below. Pick new file and click the Choose button. The old attachment will be replaced with new one.




Delete Homework

Homework assignments may be deleted. Once deleted they will automatically be removed from student and parent portal. Choosing red Delete button next to assignment will prompt a second chance question to delete. Choose red Yes, Delete button, as shown below to delete assignment.




Search Option

Searching for Homework Assignments can be done either through search box or date range. To search homework assignments for specific wording enter text in search box. Homework assignments can also be searched by Date Assigned or Due Date by clicking on the gray triangle in the search box. Simply enter dates and click on the red Search button.
