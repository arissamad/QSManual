Admissions

The Admissions Module is a simple, and convenient way to record and track potential students from their first inquiry to enrollment. Admission information can be recorded according to default parameters or customized to reflect different admissions status. In addition, Semester Management, is controlled in this module. Configure Admissions Module

An Administrator must configure the Admission Module before it can be used. Configure the Admissions Module by choosing the Features option in the menu at top of page. Then check the box next to Admissions option and click on the "Configure" link as seen below.



This pops up a window which allows administrators to list personnel who will manage admissions and run admission reports. As show below, a user selected to manage admissions will be managing all inquiries and student applications. The second box allows the administrator to list authorized personnel to run admission reports. Click red "Save" button.




Student Inquiries

Student Inquiries is an effective way to record and track potential students from first contact to application. Each new Student Inquiry is assigned an inquiry ID starting with E(Enquiry) and numbered in sequential order. Shown below is an overview of the Inquiry Page.



Add New Inquiry

Adding a new Student Inquiry is accessed by choosing "Admission" option in menu at top of page, followed by choosing the "Inquiries" tab. Then:



This brings up the "Add Student Inquiry" Screen which allows for the recording of all pertinent information relating to both the individual inquiring about the school and the potential student. To ensure consistency throughout all the modules: Student and Inquirer's Name should be entered as LAST NAME, FIRST NAME.


Add Inquiry Screen with information completed

"Inquiry Status" is a KEY element in tracking students through the admission process and in admission reports.



Edit/Update Inquiry

Student Inquiry status may be updated, comments added, status remarks included, or changes to student's information at any time. To update or edit Student Inquiry information click on the Inquiries tab in the Admissions Feature and choose the Student's name . All of the Student's information will be displayed. Any changes or updates may be recorded including Inquiry Remarks, Comments and Inquiry Status. Click red "Save" button when done.




Student Applications

Applications allows for comprehensive recording of students' personal, residential, and medical information. As with Student Inquiries, Applications track and update students' Application Status. An application may be generated from either an Inquiry or from with the Application page. The Applications Page contains the following information:


Applications are shown in order by most recent Modified Date

Convert Inquiry to Application

For consistency and to avoid duplicate records a Student Inquiry must be converted to an application. This is accomplished by choosing Inquiry Status in the Student Inquiry record and clicking on "Converted to Application". An Application record will automatically generate issuing a new Student Application Number.



Below is an overview of the Student Application. Please note when converting an Inquiry to an Application the Student ID changes from an E(Enquiry) to A(Application) and is numbered in sequential order.



New Application without Inquiry

A standalone application may be generated when no Inquiry was recorded by choosing "Applications" tab in Admissions Module then clicking on green "Add New Application" button.

Inquiry Status Codes

Inquiry Status Codes denote the state of Student Inquiries providing the ability to see at a glance where each Inquiry is in the process. There are several built-in codes and the ability to customize Status Codes as needed. These codes are used both in the Student Inquiries and the related reports. Below is an overview of the Inquiry Status Codes page.


Inquiry Status Codes with "School Visit" as a customized code.

Access the "Inquiry Status Codes" page from the Admissions feature then choose the "Inquiry Status Codes" tab as shown above. The built-in codes are default status codes and cannot be modified or deleted, only moved up or down for display purposes.


Add Inquiry Status Code

An inquiry status code may be added by accessing the "Inquiry Status Codes" page. Then:

  2. Enter new Status Code name.
  3. Click "Save"

The new inquiry status code will display as an option on the Inquiry Page, on the Student Inquiry Page, and in the Admissions Reports.



Edit/Delete Inquiry Status Codes

A customized status code may be edited or deleted, but a built-in status code cannot. To edit a status code access the Inquiry Status Page. Then:

  1. Click on the status code to be edited.
  2. Rename status code.
  3. Click "Save"

To delete a status code access the Inquiry Status Page. Then:

  1. Click on the status code to be deleted.
  2. Click Delete button as highlighted in image above.

The status code will be deleted as an option on the Inquiry Page, on the Student Inquiry, and in the Admissions Reports.

Search Admissions

The Admissions feature allows for comprehensive searching of all Inquiries and Applications. Searches may be made by status codes, names, or other criteria listed in each summary record.


Search Inquiries

To search Inquiries by Status Code:

  1. Choose "Inquiries" tab from Admissions feature.
  2. Click arrow in search box.
  3. Check box next to Inquiry Code to view (multiple boxes may be checked).
  4. Click to view records.


To search Inquiries by other criteria:

  1. Choose "Inquiries" tab from Admissions feature.
  2. Enter Search criteria (see image above for search information).
  3. Click to view records.

Search Applications

To search Applications by Status Code:

  1. Choose "Applications" tab from Admissions feature.
  2. Click arrow in search box.
  3. Check box next to Application Code to view (multiple boxes may be checked).
  4. Click to view records.


To search Applications by other criteria:

  1. Choose "Applications" tab from Admissions feature.
  2. Enter Search criteria (see image above for search information).
  3. Click to view records.
